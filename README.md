# 起岚独立博客

这是一个为 GitHub Pages 准备的静态博客版本，包含首页、文章页、RSS、站点地图和 404 页面。

## 部署到 GitHub Pages

1. 新建一个 GitHub 仓库，例如 `xiaoyaoqilan.github.io`。
2. 把本目录内容推送到仓库的 `main` 分支。
3. 在 GitHub 仓库设置里打开 Pages，选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/root`。

如果使用自定义域名，把域名写入 `CNAME` 文件并在 DNS 里添加对应记录。
